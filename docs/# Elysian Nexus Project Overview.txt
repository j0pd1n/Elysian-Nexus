# Elysian Nexus Project Overview

## Introduction
Elysian Nexus is an immersive RPG that allows players to explore a rich world filled with quests, character customization, and crafting.

## Project Structure
- **Text Rpg/**: Contains the main game logic, including menu systems and game mechanics.
- **docs/**: Contains documentation files for the project.

## Future Development
- Expand the game world with new quests and characters.
- Implement multiplayer features for cooperative gameplay.
- Enhance graphics and sound for a more immersive experience.
